# Shirutim4U - שירותים ניידים מפוארים להשכרה

A premium landing page built with Next.js and Tailwind CSS for a luxury portable toilet rental service in Israel.

## 🌟 Overview
Shirutim4U provides a high-end, hotel-quality sanitary experience for nature weddings, bar/bat mitzvahs, corporate events, and outdoor parties. This landing page is designed to highlight the luxury aspect of the service with an elegant, responsive design and premium aesthetics.

## ✨ Key Features
- **Responsive Luxury Gallery**: Custom-built media gallery featuring an auto-playing 9:16 portrait video and a 3-photo grid layout perfectly aligned to an invisible 3-column grid on desktop.
- **Perfect Media Framing**: Media containers match the native aspect ratios of smartphone cameras (3:4 for photos, 9:16 for videos) ensuring 100% of the content is visible without awkward cropping.
- **Mobile-First Swipe UI**: Native smooth-scrolling horizontal swipe gallery for mobile users.
- **Smart Contact Routing**: A dynamic "Call Now" CTA that dials the phone number directly on mobile devices, but smoothly scrolls to the contact form on desktop.
- **SEO & Copy Optimized**: Fully semantic Hebrew (RTL) structure with strategic keywords, structured data schema for LocalBusiness, and high-conversion microcopy focused on reliability and peace of mind.

## 🚀 Tech Stack
- **Framework**: Next.js 14+ (App Router)
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Language**: TypeScript & React

## 🔧 Development
To run the project locally:
```bash
npm install
npm run dev
```
Open [http://localhost:3000](http://localhost:3000) to view the application in your browser.
